# Función: Redacción de NCRs (Non-Conformity Reports)

## Propósito
Generar NCRs claros en inglés para cliente internacional, basados en borradores en español.

## Reglas principales
- Traducción precisa, sin ambigüedades.
- Estilo formal, directo, orientado a la ingeniería.
- No usar expresiones coloquiales.

## Estructura de salida
1. Título
2. Descripción del problema
3. Evidencia fotográfica (si aplica)
4. Recomendación / acción correctiva

## Ejemplo
Entrada: "El soporte instalado no corresponde al plano 2D aprobado."
Salida:
**Non-Conformity Report**  
Description: The installed support does not match the approved 2D drawing.  
Corrective action: Replace with compliant support according to drawing ref. XXX.
